# SpringBoot酒店管理系统 java792

## 项目概述

本项目是基于SpringBoot的酒店管理系统，采用前后端分离的设计模式，后端提供RESTful API与前端进行数据交互。

## 技术栈

### 后端
- SpringBoot
- MyBatis
- Thymeleaf

### 前端
- Jquery
- Bootstrap
- HTML
